# Flask 姓名搜索应用

## 项目概述
这是一个基于 Flask 框架开发的简单 Web 应用，用于在 Excel 文件中搜索姓名。用户可以在网页界面输入关键词，应用会根据关键词在 Excel 文件中查找相关姓名，并显示搜索结果。

## 功能特性
- 提供简单易用的网页搜索界面
- 支持从 Excel 文件中读取数据
- 根据用户输入的关键词进行模糊搜索

## 技术栈
- **后端**：Flask 框架
- **数据处理**：Pandas 库
